# 2026-06-20

## 创建了"灵动:视效"完整PBR材质包+专属光影项目

### 项目概述
- **项目名**: 灵动:视效 (Lively Visual Effects)
- **目标**: 为Minecraft制作一套完整的PBR（基于物理渲染）材质包及专属光影系统
- **设计参考**: 学习零雾老师（Fogg05/零雾构想）的设计理念——在不魔改原版画风的前提下，给所有方块加上真实的质感

### 创建的文件清单（130+个文件）

#### 1. 材质包 (texture_pack/) - ~112 文件
- `manifest.json` - 基岩版清单（支持RTX PBR）
- `pack.mcmeta` - Java版清单（兼容Iris/Optifine/Oculus）
- `pack_icon.png` - 程序化生成的材质包图标
- `textures/texture_list.json` - 26种方块的PBR定义
- `textures/terrain_texture.json` - 地形贴图集定义
- `texture_pack/samples/` - 10个程序化样本贴图
- `subpacks/64x/` + `subpacks/128x/` - 双分辨率PBR贴图
  - 每种方块包含5张贴图：_n(法线), _s(反射), _mer(金属/发光/粗糙), _heightmap(高度), base(color)

#### 2. 专属光影 (shader_pack/) - 18 文件
- **库文件**:
  - `lib/pbr.glsl` - Cook-Torrance BRDF + GGX NDF 完整PBR管线
  - `lib/lighting.glsl` - 体积光/大气散射/软阴影/SSAO/水面波浪
  - `lib/util.glsl` - 色调映射/色彩分级/Bloom/视差贴图等工具
- **渲染通道**:
  - `gbuffers_terrain.*` - 地形PBR渲染（输出4个GBuffer）
  - `gbuffers_water.*` - 水面渲染（波浪法线/菲涅尔/焦散）
  - `shadow.*` - 5x5 PCF软阴影
  - `composite.fsh` - 延迟光照+天空+大气散射
  - `composite1/2.fsh` - Bloom高斯模糊（水平+垂直分离）
  - `composite3.fsh` - 屏幕空间反射(SSR)
  - `composite4.fsh` - 体积雾/光
  - `final.fsh` - ACES色调映射+色彩分级+渐晕+抖动
- `shaders.properties` - 光影参数配置

#### 3. 工具 (tools/)
- `generate_pbr.py` - 完整的PBR贴图自动生成器
  - 支持30+种材质预设（零雾老师风格的材质分类）
  - 自动从颜色贴图推导法线、高度、MER、反射贴图
  - 支持单图/批量/多分辨率生成
- `requirements.txt` - Python依赖 (numpy, Pillow, scipy)

#### 4. 文档
- `README.md` - 完整中英文项目文档（项目结构/快速开始/PBR标准/材质分类）

### 零雾老师风格关键特征
- 忠实原版画风，仅增强质感和光影
- 材质物理属性统一：石质0.55-0.90粗糙度、金属0.7-1.0金属度
- 支持Java(Iris/Optifine)和Bedrock RTX双版本
- 多分辨率支持（64x/128x/256x）
- 性能优化导向

### 技术栈
- GLSL 150 (Minecraft Iris/Optifine兼容)
- Cook-Torrance BRDF + GGX NDF
- LabPBR 1.3标准 + Bedrock RTX MER格式
- Python程序化纹理生成

## 第四轮：重写着色器解决编译失败问题

### 问题诊断
- Iris 日志: `Pack "Lively-Visual-Effects" is not valid! Can't load it.`
- 原因1: `shaderpacks/` 里同时存在文件夹和zip格式，文件夹版本结构错误导致 Iris 拒绝加载
- 原因2: 原始 GLSL 使用复杂的 `#include` 依赖链和可能的编译错误
- 原因3: 缺少关键的 `gbuffers_basic` 着色器程序

### 解决方案
彻底重写所有着色器为自包含、简洁、可编译的 GLSL：
- **gbuffers_basic** (新增) - 处理大部分方块的暖冷色调光照
- **gbuffers_terrain** - PBR法线增强 + 镜面反射高光
- **gbuffers_textured** (新增) - 实体/物品光照
- **gbuffers_water** - 动态波纹 + 太阳反射 + 菲涅尔
- **composite** - 天空大气散射 + ACES色调映射 + 暗角
- **final** - Gamma校正 + 防色带抖动
- **shadow** - 基础阴影映射

### 关键改进
- 移除所有 `#include`，每个着色器自包含
- 移除 `#ifdef` 条件编译，直接使用可用功能
- 使用 `#version 150` + `ftransform()` 兼容 Iris
- 即使PBR贴图不存在也能正常渲染（降级为普通光照）
- 清理所有未使用的lib/和composite1-4文件
