# 2026-06-18

## 项目初始化 — 灵动:视效 PBR 材质包

- 搭建了 Minecraft Java 版 Iris Shaders 的 PBR 材质包项目骨架
- 许可证: GPL v3（已有）
- PBR 格式: labPBR 1.3 标准（OpenGL 法线, Y+）

### 纹理生成
- 创建 `scripts/create_textures.py` — 16 个基础贴图带 3D 浮雕效果
  - 算法: 多层噪声 + 边缘斜面 + 中心隆起 (factor^2.5~4.0)
  - 每个方块: 独立色彩 + 表面纹理 + 强烈 3D 凸起
- 方块列表: stone, cobblestone, oak_planks, grass_block_top, dirt, bricks, iron_block, gold_block, diamond_block, obsidian, glass, glowstone, netherrack, emerald_block, gravel, sand
- 批量生成 48 个贴图: 16 base + 16 normal + 16 specular
- 修复 Python 3.9 兼容性问题 (dict[str, int] → from __future__ import annotations)
- 已部署到 Minecraft 资源包目录: versions/1.21.11/resourcepacks/
- 扩展为全方块生成: 216 种方块 × 3 层 = 648 个 PBR 贴图
- 改用原版纹理: 从 1.21.11.jar 提取 1111 个原版贴图
- 生成 PBR: 1111 组 × 3 = 3333 个文件（原版纹理 + 法线 + 镜面）
- 纹理完全保持原版外观，仅添加 PBR 3D 法线+镜面效果
- 材质检测覆盖 80+ 种关键词，支持所有原版文件名

### 创建的文件
- `pack.mcmeta` — 资源包元数据（pack_format: 34）
- `README.md` — 项目说明（中英双语，含材质参数参考表）
- `docs/PBR_GUIDE.md` — 详细的 PBR 格式技术指南
- `scripts/generate_pbr.py` — PBR 贴图批量生成工具（Python）
  - 支持 26 种预设材质（石头、木头、金属、发光体等）
  - 自动检测材质类型，也可手动指定
